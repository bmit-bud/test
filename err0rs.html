Who wants to deliberately look at errors?

This Page lists all the HTTP error codes that this website has a custom html page for. While they look very similar on the surface, they all have their set of secrets. Not all of them will actually be implemented, meaning that nobody will see them. And i think that is a shame, which is the reason i made this page. Click on the Text to get to the corresponding error page or see the Mozilla Developer Network entry for the status code using the little "info" button

  • 400 - Bad Request
  • 403 - Forbidden
  • 404 - File not Found
  • 410 - Gone
  • 414 - Request-URL Too Long
  • 429 - Too Many Requests
  • 451 - Unavailable For Legal Reasons
  • 500 - Internal Server error
